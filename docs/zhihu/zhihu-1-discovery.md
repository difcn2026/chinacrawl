# 我在抖音搜"碎菜机"，结果发现了一个所有人都搞错了的品类

> 这是一篇用爬虫+数据分析写成的品类研究报告。结论先行：抖音上的"碎菜机"，99%的人以为它是厨房小家电。数据告诉我，它其实是一种养殖饲料粉碎机。

---

## 一、一个意外的发现

事情的起因很简单。我最近在研究厨房小家电赛道，想看看"碎菜机"这个品类在抖音上的竞争格局。

我打开数据采集工具，用关键词「碎菜机」在抖音上搜了一圈，抓了35个相关账号的数据。

然后我愣住了。

排名第一的账号叫「小莉创意发明」，91.6万粉丝，内容全是农业机械和养殖设备。

排名第三的叫「盛丰碎菜机工厂」，19.4万粉丝，简介写着"分体式不锈钢桶身、全铜电机、加大功率"——这显然不是厨房用的。

我继续往下翻。第四名「农具之家好物分享」，签名是"专业养殖几十年"。第六名第七名分别是「碎菜机工厂直播账号」和「碎菜机工厂直播间」。

**没有一个账号是做厨房小家电的。**

我又去淘宝和京东搜了一下"碎菜机"。结果完全相反——全是小熊、苏泊尔、荣事达的家用绞肉碎菜机。

同一个关键词，在电商平台和短视频平台指向了**两个完全不同的品类**。

---

## 二、数据怎么说？

为了验证这个发现，我用自己写的开源工具 [ChinaCrawl](https://github.com/difcn2026/chinacrawl) 做了三轮数据采集：

| 采集轮次 | 方法 | 产出 |
|---------|------|------|
| 第一轮 | SearXNG 多引擎搜索 | 电商平台品牌排行、1688批发价格 |
| 第二轮 | 抖音搜索 XHR 拦截 | 35个相关账号 + 粉丝/作品数据 |
| 第三轮 | 用户帖子 XHR 拦截 | Top10 账号各18-21条帖子互动数据 |

三轮下来，采集了35个账号、191条帖子的完整互动数据（点赞/评论/分享/收藏）。

### 35个账号的分类

| 类型 | 占比 | 代表账号 |
|------|:--:|------|
| 碎菜机工厂/厂家 | **45%** | 盛丰碎菜机工厂、碎菜机工厂直播 |
| 农具/农机相关 | 25% | 农具之家好物分享、农皇农机厂 |
| 五金/机械 | 10% | 恒锐五金制品 |
| 百货/好物推荐 | 10% | 哲哲百货推荐 |
| 创意/生活类 | 10% | 小莉创意发明 |

**将近一半的账号是工厂直销号**，另外四分之一是农具农机号。加起来，70%的内容围绕养殖/农业场景。

这意味着：**如果你是一个厨房小家电品牌，在抖音搜"碎菜机"做竞品分析——你会得出完全错误的结论。**

---

## 三、为什么不一致？

这背后的原因是平台内容生态的差异：

- **电商平台（淘宝/京东/拼多多）**: 用户搜索"碎菜机"是为了**买**。平台展示的是标准化商品。品牌方做了 SEO 优化，搜索结果自然指向厨房小家电。
- **抖音**: 用户搜索"碎菜机"可能是为了**看**。算法推荐的是用户真实发布的内容。养殖户和工厂主发的内容远多于品牌方，所以搜索结果被农业机械占据。

简单来说：电商是"货架逻辑"，抖音是"内容逻辑"。同一个关键词，在不同逻辑下指向完全不同的事物。

---

## 四、这对品牌意味着什么？

### 如果你是厨房小家电品牌

恭喜你，抖音上的厨房碎菜机赛道**几乎是一片空白**。35个相关账号中，0个做厨房赛道。

这意味着：
1. 你不需要和现有账号抢流量——因为根本没有竞争对手
2. 但你需要**换关键词**——用「绞肉机」「料理机」「电动绞肉机」而不是「碎菜机」
3. 先入为主的红利窗口大概有6-12个月

### 如果你是养殖设备厂商

你的竞争对手有16个（占45%），但好消息是——**他们都很弱**。

我分析了这35个账号的互动数据，发现了一个惊人的数字：

> **品类互动率中位数：0.35%。**
>
> 作为参考，抖音整体互动率大约在3-5%。碎菜机品类的内容质量，只有平均水平的十分之一。

而且，**91.4%的账号没有做企业认证**。这意味着只要你去认证一个蓝V，就能瞬间进入这个品类的前10%信任梯队。

## 五、如果你也想做类似的品类分析

我开源了这次分析用的全套工具：[ChinaCrawl](https://github.com/difcn2026/chinacrawl)。

它是一个面向中国互联网的开源搜索引擎+爬虫框架，对标 Firecrawl 但零外部付费依赖。支持：

- SearXNG 多引擎搜索（Google + Bing + DuckDuckGo 聚合）
- 抖音适配器（XHR拦截绕过反爬，无需破解签名算法）
- 拼多多适配器（双通道：浏览器+API降级）
- 页面抓取（Jina + Trafilatura 双引擎）

```bash
pip install chinacrawl
```

三行代码就能跑一个品类分析：

```python
from chinacrawl import search_web, scrape

# 搜索
results = search_web("碎菜机 品牌 排行", max_results=10)

# 抓取
page = scrape(results[0].url)

# 分析
print(page.title, page.word_count)
```

完整的竞品分析报告 + 复现指南在 GitHub 的 `examples/cases/` 目录下。
