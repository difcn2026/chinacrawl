项目简介

ChinaCrawl 是面向中国互联网的开源搜索引擎与数据采集引擎。 对标 Firecrawl， 但专为中文网络生态优化 —— 内置对 抖音 (Douyin) 和 拼多多 (Pinduoduo) 等平台的深度适配。

🔍 中文优先

SearXNG + Mojeek 双引擎中文搜索，原生支持百度/搜狗等中文搜索引擎 via SearXNG 配置。

💰 零付费依赖

不依赖任何外部付费 API。搜索用自建 SearXNG 实例，LLM 提取用本地 Ollama，全部自托管。

🛡️ 平台对抗

抖音 & 拼多多适配器内置反检测 (anti-detect) 引擎，绕过风控，稳定采集。

📦 11 合 1

Scrape / Map / Search / Monitor / Monitor-AI / Crawl / Download / Interact / Extract / Extract-LLM / Session。

快速开始

安装

pip install chinacrawl

依赖: httpx, trafilatura。可选: playwright (浏览器交互 & 平台适配器)。

基础用法

from chinacrawl import scrape, search_web

# 抓取单页
result = scrape("https://example.com")
print(result.title, result.content[:200])

# 搜索中文内容
results = search_web("Python 爬虫教程", max_results=10)
for r in results:
    print(r.title, r.url)

# 批量抓取
from chinacrawl import scrape_many
for r in scrape_many(["https://a.com", "https://b.com"]):
    print(r.summary)

抖音适配器

from chinacrawl import douyin_login, douyin_user_posts

douyin_login()  # 扫码登录
for post in douyin_user_posts("user_sec_uid"):
    print(post.desc, post.digg_count)

拼多多适配器

from chinacrawl import pinduoduo_product_search, pinduoduo_product_detail

for product in pinduoduo_product_search("蓝牙耳机", max_results=20):
    print(product.title, product.price)
    detail = pinduoduo_product_detail(product.goods_id)
    print(detail.description)

Core API 参考 核心

数据模型

类字段说明
ScrapeResult url, engine, title, content, content_type, elapsed_ms, error, word_count 单页抓取结果。属性: .ok, .summary
PageLink url, text, internal 站点内链接
SearchResult title, url, snippet 搜索结果条目
MonitorResult url, changed, prev_hash, curr_hash, prev_checked, curr_checked, word_count, error 页面变更检测结果
MonitorAIResult url, changed, noise_stripped, prev_hash, curr_hash, prev_checked, curr_checked, noise_removed, error AI 降噪监控结果
CrawlResult url, pages, total_pages, ok_pages, elapsed_ms, error 递归爬取结果
ExtractResult url, data, ok, elapsed_ms, error 结构化提取结果

抓取

函数签名说明
scrape (url, prefer="jina", fallback=True, **kw) → ScrapeResult 智能抓取：优先 Jina AI，失败回退 trafilatura
scrape_jina (url, timeout=TIMEOUT, proxy=PROXY) → ScrapeResult 通过 r.jina.ai 抓取为 Markdown
scrape_trafilatura (url, timeout=TIMEOUT, proxy=PROXY) → ScrapeResult 本地 trafilatura 提取正文
scrape_many (urls, engine="jina", delay=0.5) → Generator 批量抓取，逐条 yield
from chinacrawl import scrape, scrape_jina, scrape_trafilatura, scrape_many

# 默认智能抓取
r = scrape("https://httpbin.org/html")
print(r.title, r.word_count)

# 指定引擎
r = scrape_trafilatura("https://httpbin.org/html")

# 批量
for r in scrape_many(["https://a.com", "https://b.com"], delay=1.0):
    if r.ok:
        print(r.summary)

站点映射

函数签名说明
map_site (url, timeout=TIMEOUT, proxy=PROXY) → list[PageLink] 发现站点所有链接，标注 internal/external
from chinacrawl import map_site

links = map_site("https://example.com")
for link in links:
    kind = "内链" if link.internal else "外链"
    print(f"[{kind}] {link.url} — {link.text}")

搜索

函数签名说明
search_web (query, max_results=10, timeout=TIMEOUT, proxy=PROXY) → list[SearchResult] 中文搜索：Mojeek 优先 → SearXNG 回退
search_and_scrape (query, max_results=5, engine="jina", delay=0.5, **kw) → Generator 搜索 + 自动抓取搜索结果页
from chinacrawl import search_web, search_and_scrape

# 纯搜索
results = search_web("Rust 异步运行时对比", max_results=15)
for r in results:
    print(r.title, r.url)

# 搜索并抓取正文
for r in search_and_scrape("Python asyncio 最佳实践", max_results=3):
    print(r.title)
    print(r.content[:300])

监控

函数签名说明
monitor_page (url, engine="jina", label=None, timeout=TIMEOUT, proxy=PROXY) → MonitorResult Hash 变更检测，缓存到 monitor_cache/
monitor_page_ai (url, engine="jina", label=None, timeout=TIMEOUT, proxy=PROXY) → MonitorAIResult AI 降噪监控：剔除时间戳/广告等噪音后再对比
from chinacrawl import monitor_page, monitor_page_ai

# 首次运行 → changed=False (建立基线)
r = monitor_page("https://example.com/news", label="example-news")
print(r.changed, r.prev_hash, r.curr_hash)

# 页面更新后再次运行 → changed=True

# AI 降噪版本（剔除时间戳等动态内容）
r = monitor_page_ai("https://example.com/news")
print(r.changed, r.noise_removed)

爬取

函数签名说明
crawl_site (url, max_depth=3, max_pages=100, path_prefix=None, ...) → CrawlResult 深度限制递归爬取，默认 trafilatura 提取
download_site (url, output_dir, max_pages=50, engine="jina", delay=0.5, **kw) → CrawlResult Map + Scrape → 保存 Markdown 到磁盘
from chinacrawl import crawl_site, download_site

# 递归爬取
result = crawl_site("https://docs.example.com", max_depth=2, max_pages=50)
print(f"抓取 {result.ok_pages}/{result.total_pages} 页，耗时 {result.elapsed_ms:.0f}ms")

# 下载整站到本地
result = download_site("https://docs.example.com", output_dir="./example_docs/")
print(f"已保存到 ./example_docs/")

浏览器交互

函数签名说明
browser_interact (url, actions, timeout=TIMEOUT, proxy=PROXY, headless=True) → dict Playwright 浏览器自动化：点击/输入/截图/等待
browser_session_save (name, url="", timeout=TIMEOUT, proxy=PROXY, headless=True) → dict 保存浏览器登录态
browser_session_load (name, timeout=TIMEOUT, proxy=PROXY, headless=True) → dict 加载已保存的登录态
browser_session_close (session_result) → None 关闭浏览器会话
from chinacrawl import browser_interact

result = browser_interact("https://example.com/login", actions=[
    {"type": "fill",   "selector": "#username", "value": "admin"},
    {"type": "fill",   "selector": "#password", "value": "pass123"},
    {"type": "click",  "selector": "button[type=submit]"},
    {"type": "wait",   "ms": 2000},
    {"type": "content"},
    {"type": "screenshot", "path": "./login_result.png"},
])
print(result["content"][:500])

结构化提取

函数签名说明
extract_structured (url, schema, timeout=TIMEOUT, proxy=PROXY) → ExtractResult 基于 schema 字典的 CSS 选择器提取
extract_llm (url, schema, model="qwen2.5:7b", content=None, timeout=TIMEOUT, proxy=PROXY) → ExtractResult LLM 语义提取 (需本地 Ollama)
from chinacrawl import extract_structured, extract_llm

# CSS 选择器提取
schema = {
    "title":     "h1",
    "price":     ".price-tag",
    "rating":    ".star-rating | attr:data-score",
    "reviews":   ".review-item | [text]",
}
result = extract_structured("https://example.com/product", schema)
print(result.data)  # {"title": "...", "price": "...", ...}

# LLM 语义提取（需要 Ollama 运行中）
schema = {
    "product_name": "string",
    "price": "number",
    "pros": ["string"],
    "cons": ["string"],
}
result = extract_llm("https://example.com/review", schema, model="qwen2.5:7b")
print(result.data)

Douyin Adapter API 抖音

数据模型

类说明
UserInfo用户信息：昵称、头像、粉丝数、获赞数、sec_uid
AwemeInfo视频信息：描述、点赞、评论、分享、播放数、下载地址
CommentInfo评论信息：内容、点赞数、回复数、用户
SearchResult搜索结果：类型、标题、URL、附加数据

用户操作

函数签名
user_info(sec_uid: str) → UserInfo
user_posts(sec_uid: str, max_pages: int = 0, ...) → Generator[AwemeInfo]
user_likes(sec_uid: str, max_pages: int = 0) → Generator[AwemeInfo]
from chinacrawl import douyin_user_info, douyin_user_posts, douyin_user_likes

user = douyin_user_info("MS4wLjABAAAA...")  # sec_uid
print(f"{user.nickname} — {user.follower_count} 粉丝")

for post in douyin_user_posts("MS4wLjABAAAA...", max_pages=3):
    print(post.desc, post.digg_count)

视频操作

函数签名
video_info(aweme_id: str) → AwemeInfo
video_download(aweme_id: str, output_dir="./downloads/", ...) → str
video_comments(aweme_id: str, max_pages: int = 5) → Generator[CommentInfo]
from chinacrawl import douyin_video_info, douyin_video_download, douyin_video_comments

info = douyin_video_info("7123456789012345678")
path = douyin_video_download("7123456789012345678", output_dir="./videos/")
print(f"已下载: {path}")

for comment in douyin_video_comments("7123456789012345678"):
    print(f"{comment.user_name}: {comment.text}")

搜索

函数签名
search(keyword: str, max_results: int = 20) → List[SearchResult]
search_user(keyword: str, max_results: int = 10) → List[UserInfo]
search_hashtag(keyword: str, max_results: int = 10) → List[dict]
from chinacrawl import douyin_search, douyin_search_user, douyin_search_hashtag

# 搜索视频
for r in douyin_search("猫咪", max_results=20):
    print(r.title, r.url)

# 搜索用户
for u in douyin_search_user("美食博主"):
    print(u.nickname, u.follower_count)

# 搜索话题
for h in douyin_search_hashtag("旅行"):
    print(h["title"], h["view_count"])

会话管理

函数签名

Pinduoduo Adapter API 拼多多

数据模型

类说明
ProductInfo商品详情：标题、价格、销量、描述、图片、规格
ShopInfo店铺信息：名称、评分、销量、关注数
ReviewInfo评价信息：内容、评分、图片、用户
SearchResult搜索结果项：商品标题、价格、销量、goods_id

商品操作

函数签名
product_search(keyword: str, max_results: int = 20, ...) → SearchResult 列表
product_detail(goods_id: str, cookie_file: Optional[str] = None) → Optional[ProductInfo]
product_reviews(goods_id: str, max_reviews: int = 50, ...) → ReviewInfo 列表
product_download(goods_id: str, output_dir="./downloads/", ...) → dict
from chinacrawl import (
    pinduoduo_product_search, pinduoduo_product_detail,
    pinduoduo_product_reviews, pinduoduo_product_download
)

# 搜索商品
for p in pinduoduo_product_search("蓝牙耳机", max_results=20):
    print(f"{p.title} — ¥{p.price} — 销量 {p.sales}")

# 商品详情
detail = pinduoduo_product_detail("12345678")
print(detail.title, detail.description, detail.specs)

# 商品评价
for r in pinduoduo_product_reviews("12345678", max_reviews=50):
    print(r.rating, r.content)

# 下载商品信息
info = pinduoduo_product_download("12345678", output_dir="./pdd_products/")

部署指南

🏠 本地开发

pip install chinacrawl
# 可选：Playwright 浏览器
playwright install chromium

直接导入使用，无需额外配置。

🖥️ 生产部署

# 1. 自建 SearXNG 实例 (Docker)
docker run -d -p 8888:8080 searxng/searxng

# 2. 配置代理 (推荐)
export HTTP_PROXY=http://proxy:7654

# 3. 安装 Ollama (可选，用于 extract_llm)
curl -fsSL https://ollama.com/install.sh | sh
ollama pull qwen2.5:7b

☁️ 平台适配器

# 抖音 & 拼多多需要 Playwright
pip install playwright
playwright install chromium

# 登录一次保存 cookie
python -c "
from chinacrawl import douyin_login, douyin_save_session
douyin_login()
douyin_save_session('./dy_cookies.json')
"

FROM python:3.12-slim
