# Contributing to xhls_scraper

Welcome! xhls_scraper is an open-source, China-optimized web data engine under AGPLv3.

## How to contribute

### Report bugs

Use the [Bug Report](https://github.com/xiaohei/xhls_scraper/issues/new?template=bug_report.md) template.

### Suggest features

Use the [Feature Request](https://github.com/xiaohei/xhls_scraper/issues/new?template=feature_request.md) template.

### Submit code

1. Fork the repository
2. Create a branch: `feature/your-feature` or `fix/your-fix`
3. Write your changes
4. Add tests if applicable
5. Run the existing tests: `python xhls_scraper.py`
6. Submit a Pull Request

## Code style

- Follow PEP 8
- Use type hints where practical
- Keep the single-file philosophy (new capabilities = new functions, not new files)
- Chinese comments OK for complex logic, English preferred for API docs

## License

By contributing, you agree that your contributions will be licensed under AGPLv3.

## Questions?

Open a [Discussion](https://github.com/xiaohei/xhls_scraper/discussions) or email the maintainer.
