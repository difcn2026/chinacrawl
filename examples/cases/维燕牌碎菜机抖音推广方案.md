## 二、品牌定位：维燕是谁？

### 2.1 赛道决策

维燕应同时布局 **双赛道**，但以厨房赛道为旗舰、养殖赛道为侧翼：

| 赛道 | 定位 | 账号名 | 优先级 |
|------|------|--------|:--:|
| 🏠 **厨房** | 家用电动碎菜料理机 | `维燕厨房` | P0 旗舰 |
| 🐔 **养殖** | 商用/养殖饲料粉碎机 | `维燕碎菜机工厂` | P1 侧翼 |

> **决策理由**: 厨房赛道 0 竞品、高增长（47.15%）、年轻女性受众 —— 是品牌溢价的主战场。养殖赛道有现成需求但竞争者多（45%），以工厂身份切入做价格锚点和信任底座。

### 2.2 品牌人设

```
品牌名: 维燕 (WeiYan)
人设: 「懂电机也懂食材的工程师品牌」
调性: 专业但不冷冰 / 真实但不粗糙 / 数据驱动但不枯燥
对标: 小熊的颜值 × 盛丰的工厂真实感 × 九阳的技术流
Slogan: 「碎得讲究」
```

### 2.3 品牌视觉规范

| 元素 | 规范 |
|------|------|
| 主色 | 燕羽白 #F5F0E8 + 深空灰 #2D2D2D |
| 辅色 | 食材绿 #5B8C5A（碎菜机内容关联色） |
| 字体 | 视频标题: 思源黑体 Bold / 数据标注: DIN |
| 拍摄风格 | 自然光 + 顶灯补光，避免过度滤镜 |
| 视频比例 | 9:16 竖屏（抖音原生） |

#### 🐔 侧翼号：`维燕碎菜机工厂`

| 要素 | 策略 |
|------|------|
| 定位 | 源头工厂品牌直销 |
| 目标人群 | 养殖户、农场主、餐饮采购 |

### 3.2 账号注册清单

| 步骤 | 行动 | 完成标准 |
|:--:|------|------|
| 1 | 注册两个抖音账号 | 手机号+营业执照准备 |
| 2 | **立即企业认证**（双号） | 蓝V标识上线 |
| 3 | 开通抖音小店 | 上架 3-5 个 SKU |

> ⚠️ **关键**: 2 个账号的企业认证必须在上线前完成。91% 的对手没认证，这是维燕的第一个护城河。

### 4.3 内容制作 SOP

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 选题会    │ →  │ 脚本撰写  │ →  │ 拍摄+剪辑 │ →  │ 发布+运营 │
│ 每周日晚  │    │ 周一上午  │    │ 周二-周四 │    │ 周五-周日 │
│ 30min    │    │ 60min    │    │ 3h       │    │ 持续      │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
```

**单条视频制作标准**:
- 前 3 秒钩子（参考数据：品类 top 视频均在前 3 秒展示机器运转）
- 时长 25-45 秒（抖音最佳带货时长区间）
- 字幕: 黄底黑边硬字幕（工厂类账号验证有效）
- 音乐: 轻快节奏（避免土味BGM，拉开与白牌差距）

### 4.4 钩子公式库

基于竞品爆款分析，以下钩子类型在碎菜机品类有效：

| 类型 | 示例 | 数据支撑 |
|------|------|---------|
| **效果反差** | "以前剁菜 20 分钟，现在 10 秒" | 农具之家 top 帖 52K 赞 |
| **价格锚点** | "工厂价 ¥199，同配置别家卖 ¥499" | 工厂直播类高转化 |
| **材质揭秘** | "拆开给你看，这里面到底是铜还是铝" | 品类无竞品做，独占空白 |
| **场景共鸣** | "带娃还要做饭？这个救了命了" | 宝妈场景高互动 |
| **数据说话** | "连续碎 30 分钟，温度只升了 8 度" | 九阳技术流验证有效 |

## 七、转化漏斗设计

```
短视频曝光
    ↓ CTR 3-5%
主页/小黄车访问
    ↓ 转化率 2-3%
商品详情页
    ↓ 加购率 8-12%
直播间成交 / 静默下单
    ↓
私域沉淀（企业微信）
    ↓ 复购率 15-20%
老客带新（裂变）
```

### 7.1 直播间 SOP

| 环节 | 时长 | 内容 |
|------|:--:|------|
| 暖场 | 5 min | 机器空转展示 + 欢迎语 |
| 产品讲解 | 10 min | 参数 + 材质 + 价格对比 |
| 演示 | 10 min | 现场碎菜 + 清洗展示 |
| 逼单 | 5 min | 「今天下单送 xx」限时优惠 |
| 答疑 | 10 min | 回答弹幕问题 |
| 循环 | — | 重复 2-5 轮 |

**工厂号直播间核心指标**:
- 场均观看 > 500
- 在线峰值 > 50
- 转化率 > 2%
- 客单价 ¥199-399
