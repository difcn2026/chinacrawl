# 剧本审查技能 · Script Review

审查剧本文件的结构完整性、角色一致性、时长合理性和平台适配性。

## 触发条件
- 用户要求 "审查剧本"、"检查剧本"、"review script"
- 剧本创作阶段完成后自动触发

## 审查维度
1. **结构检查** — 是否有钩子、铺垫、升级、反转、反应、收尾、下集钩子
2. **时长估算** — 每个分镜时长是否在合理范围，总时长是否符合目标
3. **角色一致性** — 角色行为和台词是否符合人设
4. **平台适配** — 字幕密度、节奏是否适合目标平台

## 输出格式
```
## 剧本审查报告
- 文件：{path}
- 总分：X/100

### 结构 (X/30)
- 问题1
- 建议1

### 时长 (X/25)
- ...

### 角色 (X/25)
- ...

### 平台 (X/20)
- ...
```
