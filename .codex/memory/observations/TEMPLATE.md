# 观察记录模板 · Observation Record Template

## 格式说明
每次完成任务或审查后，记录一条有价值的观察。

## 模板
```
# 观察记录：{简短标题}

- **日期**：YYYY-MM-DD
- **来源任务**：{任务名}
- **类别**：模式发现 / 痛点 / 亮点 / 规律

## 观察内容
{描述观察到的现象}

## 影响
{这个观察对后续工作意味着什么}

## 建议
{如果有的话 — 可以转化为 learning-rules 的条目}
```

## 命名规范
`YYYY-MM-DD-{short-slug}.md`

示例：`2026-06-03-douyin-audience-pattern.md`
