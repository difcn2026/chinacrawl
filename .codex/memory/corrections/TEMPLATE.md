# 纠正记录模板 · Correction Record Template

## 格式说明
每次发现并修复一个错误或不足时，按此模板创建一个文件。

## 模板
```
# 纠正记录：{简短标题}

- **日期**：YYYY-MM-DD
- **来源**：{哪个任务 / 哪个审查暴露的}
- **严重程度**：P0 阻断 / P1 重要 / P2 优化

## 问题描述
{一句话描述问题}

## 根因
{为什么发生}

## 修复方案
{做了什么修改}

## 预防措施
{如何避免再次发生 — 是否更新了规则、技能或 AGENTS.md}
```

## 命名规范
`YYYY-MM-DD-{short-slug}.md`

示例：`2026-06-03-script-timing-overflow.md`
